# Plan: Remove Image Cache

## Summary

Remove the image cache from `src/utils.ts` and all its usages. The cache stores compressed image results keyed by `providerID:dataURI` to avoid re-compressing the same image. It's not useful: sharp compression is fast (~50-200ms), the cache is in-memory only (lost on restart), grows unbounded despite a MAX_CACHE_SIZE eviction, and adds unnecessary complexity.
